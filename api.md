`Bars` (component)
==================

Bars : draws a bar graph consisting of a number of bars

Props
-----

### `barBackgroundColor`

type: `string`
defaultValue: `'#cccccc'`


### `barColor`

type: `string`
defaultValue: `'#303030'`


### `barHeight`

type: `number`
defaultValue: `42`


### `data` (required)

type: `array`


### `label`

type: `string`
defaultValue: `''`


### `makeUppercase`

type: `bool`
defaultValue: `false`


### `maxValue`

type: `number`
defaultValue: `100`


### `showValue`

type: `bool`
defaultValue: `true`


### `value`

type: `number`
defaultValue: `0`


### `verticalSpacing`

type: `number`
defaultValue: `46`
